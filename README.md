# 【唯客學院微課程】樹莓派(Raspberry Pi)使用簡介

### 課程內容：

* 課程簡介
* 燒錄OS
* Headless Connection
* SSH簡介 (使用內建SSH或putty)
* 遠端開發 (Remote Development)
* 從 RPi 發送 Line Notify 訊息
* VNC 連線 (圖形化介面)
* MQTT簡介
* 設定發送(Publish)訊息至 MQTT Broker
* 設定接收(Subscribe)自 MQTT Broker 的訊息
* 使用 DHT11, 22 量測溫溼度 (安裝adafruit_circuitpython套件)
